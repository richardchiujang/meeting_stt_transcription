========================================
AI 語音轉文字工具 - 使用說明
========================================

版本：2.0
更新日期：2026-04-13

本工具是離線語音轉文字應用程式，使用 OpenAI Whisper / Faster-Whisper 模型，
支援錄製麥克風/系統音、選取影音檔案進行中英文混合轉錄。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
一、主要功能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ 錄音功能
  - 麥克風錄音：錄製環境音、會議發言
  - 系統音錄音：錄製電腦播放的聲音（網路會議、影片等）
  - 即時音量顯示

✓ 轉錄功能
  - 即時轉錄：邊錄邊轉，即時顯示文字（約 2.5 秒延遲）
  - 批次轉錄：錄完/選檔後一次性轉錄
  - 支援多種格式：mp3, wav, mp4, mkv, m4a, mov, wmv

✓ 語言優化
  - 主要中文：適合中文為主、夾雜英文術語的內容
  - 主要英文：適合英文為主的技術會議
  - 系統自動判斷：由 AI 自動偵測語言

✓ 模型選擇
  - Faster-Whisper：速度快，推薦使用
  - OpenAI Whisper：原版實作，較慢但穩定

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
二、使用步驟
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【錄音模式】

1. 選擇設定
   □ 模型：faster-whisper-base (建議新手) 或其他
   □ 語言模式：主要中文 / 主要英文 / 系統自動判斷
   □ 錄音來源：麥克風 / 系統音

2. 選擇轉錄方式
   □ 勾選「即時轉錄」→ 邊錄邊轉，即時顯示
   □ 不勾選 → 錄完後再轉錄

3. 開始錄音
   → 點擊「開始錄音」按鈕
   → 對著麥克風說話 (或播放系統音)
   → 觀察音量條確認有收音

4. 停止錄音
   → 點擊「開始錄音」按鈕（再次點擊）
   → 即時模式：立即停止並保存
   → 非即時模式：點擊「停止錄音並轉錄」

5. 查看結果
   → 轉錄結果顯示在「轉錄結果」視窗
   → 文字檔自動儲存至 exports/ 資料夾
   → 錄音檔儲存至 recordings/ 資料夾

【選檔案模式】

1. 選擇設定
   □ 模型：同上
   □ 語言模式：同上

2. 選擇檔案
   → 點擊「選取檔案 (音/影)」按鈕
   → 選擇要轉錄的音訊或影片檔案

3. 選擇轉錄方式
   □ 勾選「即時轉錄」→ 分段處理，顯示進度
   □ 不勾選 → 整檔處理，一次顯示

4. 等待轉錄
   → 進度條顯示轉錄進度
   → 即時模式：逐段顯示文字
   → 批次模式：完成後一次顯示

5. 查看結果
   → 同錄音模式

【其他功能】

□ 清除：清空「轉錄結果」視窗
□ 停止轉錄：中斷進行中的轉錄（部分結果會保存）

三、模型說明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【Faster-Whisper vs OpenAI Whisper】

Faster-Whisper（推薦）：
  ✓ 速度：比 OpenAI Whisper 快 4-5 倍
  ✓ 記憶體：用量較少
  ✓ 準確度：與原版相當
  ✓ 檔案格式：使用 .bin 或 .safetensors

OpenAI Whisper（原版）：
  ○ 速度：較慢
  ○ 記憶體：用量較大
  ○ 準確度：穩定可靠
  ○ 檔案格式：使用 .pt

【模型大小比較】

┌──────────┬──────┬────────┬──────────┐
│ 模型     │ 大小 │ 速度   │ 準確度   │
├──────────┼──────┼────────┼──────────┤
│ base     │ 150M │ 最快   │ 基本     │
│ small    │ 500M │ 中等   │ 良好     │
│ medium   │ 1.5G │ 較慢   │ 優秀     │
└──────────┴──────┴────────┴──────────┘

建議選擇：
  • 測試/快速轉錄：base
  • 日常使用：small
  • 高品質需求：medium

【語言模式說明】

主要中文：
  - 適用：中文會議、夾雜英文術語
  - 設定：language='zh', prompt="以繁體中文為主，夾雜英文術語。"
  - 效果：優先識別中文，英文術語也能正確顯示

主要英文：
  - 適用：英文技術會議、簡報
  - 設定：language='en', prompt="This is a technical meeting in English."
  - 效果：優先識別英文，拼字更準確

系統自動判斷：
  - 適用：不確定主要語言、多語混合
  - 設定：language=None, prompt="繁體中文與英文混合討論。"
  - 效果：AI 自動判斷，但可能不如指定語言準確

四、系統音錄音 (Loopback)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

用途：錄製電腦播放的聲音（網路會議、影片、音樂等）

設定步驟：
1. 錄音來源選擇「系統音」
2. 確認裝置清單有 loopback 裝置（如「喇叭 (loopback)」）
3. 播放測試音訊，確認音量條有反應

注意事項：
• Windows 會使用 WASAPI loopback 技術
• 部分裝置可能顯示為「立體聲混音」或「Stereo Mix」
• 若無法錄製，請檢查音效卡驅動程式是否支援 loopback

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
五、檔案位置
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

錄音檔：recordings/
  • 格式：wav (16kHz mono)
  • 命名：sys_MMDD_HHMM.wav (系統音) / mic_MMDD_HHMM.wav (麥克風)

轉錄文字：exports/
  • 格式：txt (UTF-8)
  • 命名：note_MMDD_HHMM.txt
  • 內容：包含會議摘要 prompt + 轉錄文字

模型檔案：model/
  • faster-whisper/ - Faster-Whisper 模型
  • whisper/ - OpenAI Whisper 模型
  • 注意：模型檔案較大，不會自動下載

六、進階功能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【即時轉錄原理】

錄音時：
  → 每 0.2 秒產生一個音訊片段
  → 累積到 2.5 秒後送入 Whisper 轉錄
  → 轉錄結果即時顯示

優點：即時反饋，適合會議記錄
缺點：約 2.5 秒延遲

【批次轉錄原理】

錄音完成後：
  → 整段音訊一次性處理
  → 轉錄完成後一次顯示所有文字

優點：準確度較高，適合正式文件
缺點：需等待錄音完成

【會議摘要功能】

轉錄完成後：
1. 打開 exports/ 資料夾
2. 找到最新的 note_*.txt 檔案
3. 將整個檔案內容複製到 GitHub Copilot 或 ChatGPT
4. AI 會自動產生會議摘要（包含議題、討論、待辦事項）

七、常見問題
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1: 轉錄結果語言錯誤？
A1: 確認「語言模式」選擇正確：
    • 中文內容 → 選「主要中文」
    • 英文內容 → 選「主要英文」

Q2: 系統音錄不到聲音？
A2: 檢查：
    • 錄音來源是否選擇「系統音」
    • 是否有 loopback 裝置
    • 播放音訊時音量條是否有反應

Q3: 即時轉錄延遲太久？
A3: 2.5 秒延遲是正常設計，確保轉錄品質。
    若需要更快反應，可選擇「批次轉錄」。

Q4: 轉錄結果出現奇怪的文字？
A4: 可能是 AI 輸出 prompt 內容，新版已自動過濾。
    若仍出現，請選擇正確的「語言模式」。

Q5: 找不到模型檔案？
A5: 確認 model/ 資料夾存在，且包含：
    • faster-whisper/faster-whisper-base/config.json
    • 其他模型檔案

Q6: 程式啟動很慢？
A6: 首次啟動需載入模型，約 10-30 秒屬正常。
    後續啟動會較快。

Q7: 如何更新程式？
A7: 關閉程式，下載新版檔案，覆蓋舊檔即可。
    model/ 資料夾可保留不變。

八、技術規格
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

支援格式：
  • 音訊：mp3, wav, m4a
  • 影片：mp4, mkv, mov, wmv

音訊處理：
  • 採樣率：16kHz (自動轉換)
  • 聲道：單聲道 (自動轉換)
  • 編碼：PCM (WAV)

硬體需求：
  • CPU：Intel i5 或以上 (CPU 模式)
  • GPU：NVIDIA RTX 系列 (CUDA 模式，可選)
  • 記憶體：4GB 以上 (base 模型) / 8GB 以上 (medium 模型)
  • 硬碟：5GB 以上 (含模型)

軟體需求：
  • 作業系統：Windows 10/11
  • Python：3.10 或以上 (開發模式)
  • 執行檔：免安裝，解壓縮即可使用
